Sistema de Scraping para Mister/Fantasy

Python Web Scraping Automatización Análisis de Datos

Solución de web scraping desarrollada en Python para extraer y analizar estadísticas de jugadores de fútbol en la plataforma Mister/Mundodeportivo. El sistema automatiza la recolección de datos por jornada, permitiendo un seguimiento detallado del rendimiento de los jugadores en cada partido.

Demostración del Sistema

Demostración del sistema en funcionamiento

Descripción del Proyecto

Solución de web scraping desarrollada en Python para extraer y analizar estadísticas de jugadores de fútbol en la plataforma Mister/Mundodeportivo. El sistema automatiza la recolección de datos por jornada, permitiendo un seguimiento detallado del rendimiento de los jugadores en cada partido.

Características Principales
Extracción de Datos
  • Obtención automática de estadísticas de jugadores
  • Captura de datos por jornada
  • Detección de jugadores que no han jugado
  • Identificación de posiciones vacías
Generación de Informes
  • Exportación a Excel organizada por jornada
  • Resumen de estadísticas por equipo
  • Historial de rendimiento
Análisis por Jornada
  • Seguimiento de puntos por jugador
  • Registro de goles (incluyendo penaltis)
  • Control de tarjetas (amarillas, rojas, doble amarilla)
  • Cálculo de asistencias
Tecnologías Utilizadas
Backend
  • Python 3.13
  • Selenium WebDriver
  • OpenPyXL
  • WebDriver Manager
Funcionalidades Técnicas
  • Navegación automatizada en sitios web
  • Manejo de elementos dinámicos con WebDriverWait
  • Procesamiento y limpieza de datos
  • Generación de informes estructurados
Estructura del Código
  • Extracción de datos de jugadores que han jugado
  • Detección de jugadores que no han jugado
  • Identificación de posiciones vacías
  • Procesamiento de estadísticas detalladas
  • Generación de informes en formato Excel
El sistema está diseñado para ser ejecutado periódicamente, actualizando las estadísticas según avanza la temporada. Los informes generados permiten un análisis detallado del rendimiento de los jugadores a lo largo de las diferentes jornadas.

Sistema de Automatización de Tareas en Redes Sociales con UiPath

RPA UiPath Automatización

Solución RPA para automatizar la extracción y análisis de datos de redes sociales utilizando UiPath.

Capturas del Proyecto

Interfaz de UiPath Studio

Estructura del proyecto

Tweets procesados

Descripción del Proyecto

Desarrollo de una solución RPA para la automatización de tareas en redes sociales, enfocada en la extracción y análisis de datos de Twitter. El proyecto incluye la generación automática de informes detallados con métricas clave.

Características principales:
  • Automatización de extracción de datos de Twitter
  • Generación de informes estructurados en formato Excel
  • Configuración de rutas personalizadas para guardar informes
  • Procesamiento de grandes volúmenes de datos
Estructura del Informe Generado:
Campo Descripción
ID del Tweet Identificador único de cada publicación
URL del Tweet Enlace directo a la publicación
Usuario Nombre del usuario que publicó el tweet
Fecha de Publicación Marca de tiempo de la publicación
Contenido Texto completo del tweet
Métricas Likes, retweets, respuestas, etc.
Tecnologías utilizadas:
  • UiPath Studio
  • Excel para generación de informes
  • Selectores avanzados de UiPath
  • Manejo de datos estructurados
